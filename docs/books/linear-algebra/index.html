Chapter One · The building block

Vectors & Vector Spaces

A vector is the simplest object in linear algebra — an ordered tuple of numbers. For a data scientist it is everything: a single row of a dataset, an image flattened into pixels, a document's TF–IDF representation, a gradient during training.

1.1 Scalars, vectors, and arithmetic

Definition · Vector in $\mathbb{R}^n$
A vector $\mathbf{v}\in\mathbb{R}^{n}$ is an ordered tuple $\mathbf{v}=(v_1,v_2,\dots,v_n)^{\top}$ where each $v_i\in\mathbb{R}$. We add vectors component-wise and scale them by a scalar $\alpha\in\mathbb{R}$: $$\mathbf{u}+\mathbf{v}=(u_1+v_1,\dots,u_n+v_n)^{\top},\qquad \alpha\mathbf{v}=(\alpha v_1,\dots,\alpha v_n)^{\top}.$$

Addition is commutative and associative, scaling distributes over addition, and the zero vector $\mathbf{0}$ is the identity. Together, these 8 axioms define a vector space.

1.2 Dot product and norm

The dot product measures how aligned two vectors are:

$$\langle\mathbf{u},\mathbf{v}\rangle = \mathbf{u}^{\top}\mathbf{v} = \sum_{i=1}^{n} u_i v_i = \|\mathbf{u}\|\,\|\mathbf{v}\|\,\cos\theta.$$(1.1)

The Euclidean norm $\|\mathbf{v}\|=\sqrt{\langle\mathbf{v},\mathbf{v}\rangle}$ is its length. Two vectors are orthogonal when $\langle\mathbf{u},\mathbf{v}\rangle=0$ and parallel when $|\cos\theta|=1$.

1.3 Geometry of vectors

1.4 Worked example

Let $\mathbf{u}=(2,2)^{\top}$ and $\mathbf{v}=(3,1)^{\top}$.

  • $\mathbf{u}+\mathbf{v}=(5,3)^{\top}$.
  • $\langle\mathbf{u},\mathbf{v}\rangle = 2\cdot 3 + 2\cdot 1 = 8$.
  • $\|\mathbf{u}\|=\sqrt{8}\approx 2.83$, $\|\mathbf{v}\|=\sqrt{10}\approx 3.16$.
  • $\cos\theta = 8/(\sqrt{8}\sqrt{10}) \approx 0.894 \Rightarrow \theta\approx 26.6^{\circ}$.
Data Scientist Lens
Cosine similarity in NLP, recommender systems, and image retrieval is literally $\cos\theta$ from equation (1.1). Every sentence-embedding API you have ever called computes vector dot products and norms under the hood. 
import numpy as np
u = np.array([2, 2]); v = np.array([3, 1])
print(u + v)                    # [5 3]
print(u @ v)                    # 8   (dot product)
print(np.linalg.norm(u))        # 2.828...
cos = (u @ v) / (np.linalg.norm(u) * np.linalg.norm(v))
print(np.degrees(np.arccos(cos)))  # 26.56...

Q. Two vectors $\mathbf{u},\mathbf{v}\in\mathbb{R}^{3}$ satisfy $\|\mathbf{u}\|=3,\;\|\mathbf{v}\|=4$ and $\mathbf{u}\cdot\mathbf{v}=0$. What is $\|\mathbf{u}+\mathbf{v}\|$?

  1. $1$
  2. $5$
  3. $7$
  4. $\sqrt{7}$
Answer: B. Orthogonality gives $\|\mathbf{u}+\mathbf{v}\|^{2}=\|\mathbf{u}\|^{2}+\|\mathbf{v}\|^{2}=9+16=25$ (Pythagoras in $\mathbb{R}^{n}$).

Given $\mathbf{a}=(1,-2,2)^{\top}$ and $\mathbf{b}=(3,0,-4)^{\top}$, compute (i) $\mathbf{a}+\mathbf{b}$, (ii) $\mathbf{a}\cdot\mathbf{b}$, (iii) the angle $\theta$ between them, and (iv) a unit vector in the direction of $\mathbf{a}$.

A 3-feature data point is $\mathbf{x}=(0.5, -1.2, 3.0)^{\top}$ and a learned weight vector is $\mathbf{w}=(2,-1,0.5)^{\top}$. Compute the model's linear score $\mathbf{w}^{\top}\mathbf{x}+b$ for bias $b=-0.25$. State the predicted class if the decision rule is $\hat{y}=1$ when the score $>0$.

Recap Card

  • Vectors add component-wise; scalars stretch them.
  • Dot product $= \|\mathbf{u}\|\|\mathbf{v}\|\cos\theta$; orthogonal iff zero.
  • Norm $=\sqrt{\mathbf{v}^{\top}\mathbf{v}}$; unit vector $\hat{\mathbf{v}}=\mathbf{v}/\|\mathbf{v}\|$.
  • Cosine similarity, linear model scores, and gradient steps are all dot products.

Chapter Two · Tables of numbers that compute

Matrices & Matrix Operations

A matrix is a rectangular grid of numbers. For the data scientist, a dataset of $m$ samples with $n$ features is an $m\times n$ matrix $X$. Matrices are not just storage — they are functions: multiplying by a matrix transforms one vector space into another.

2.1 Matrix algebra

Definition · $m\times n$ Matrix
$A\in\mathbb{R}^{m\times n}$ is a grid with entries $a_{ij}$ ($i\in\{1..m\}$ rows, $j\in\{1..n\}$ columns). Addition and scaling are component-wise. The product $AB$ is defined when the inner dimensions match ($A$ is $m\times k$, $B$ is $k\times n$) and yields an $m\times n$ matrix with $(AB)_{ij}=\sum_{\ell=1}^{k}a_{i\ell}b_{\ell j}$.

Matrix multiplication is associative ($A(BC)=(AB)C$), distributive over addition, but not commutative: $AB\neq BA$ in general.

2.2 Matrix–vector multiplication, two views

$A\mathbf{x}$ can be read two ways:

  1. Row view. The $i$-th entry of $A\mathbf{x}$ is the dot product of row $i$ of $A$ with $\mathbf{x}$.
  2. Column view. $A\mathbf{x}=x_1\mathbf{a}_1+x_2\mathbf{a}_2+\dots+x_n\mathbf{a}_n$ — a linear combination of the columns of $A$. (This is the single most useful picture in linear algebra.)

2.3 Special matrices

MatrixDefinitionWhy a data scientist cares
Identity $I_n$$I_{ij}=1$ if $i=j$, else $0$Initialisation; "no-op"
Diagonal $D$Only $d_{ii}\neq 0$Feature scaling, SVD singular values
Symmetric $A=A^{\top}$Mirror across the diagonalCovariance, kernels, Hessians
Orthogonal $Q$$Q^{\top}Q=I$Rotations; numerical stability
Permutation $P$Single 1 per row/colRe-ordering rows/columns

2.4 Transpose and inverse (preview)

$(A^{\top})_{ij}=a_{ji}$. Useful identities: $(AB)^{\top}=B^{\top}A^{\top}$ and, when the inverse exists, $(AB)^{-1}=B^{-1}A^{-1}$. The inverse satisfies $A^{-1}A=AA^{-1}=I$ and is only defined for square, full-rank matrices — a subject we return to in Chapter 4.

Data Scientist Lens
In a neural network's dense layer y = Wx + b, $W$ is a matrix, $\mathbf{x}$ a batch of activations, and training means nudging the entries of $W$ in the direction of the negative gradient. Understanding how $W$ re-combines inputs is understanding what the layer learns. 
import numpy as np
A = np.array([[1, 2, 0],
              [0, 1, 3],
              [2, 0, 1]])
x = np.array([1, 2, 1])
print(A @ x)         # [5 5 3]
print(A.T)           # transpose
print(np.linalg.matrix_rank(A))  # 3 (full rank)

Q. If $A\in\mathbb{R}^{3\times 4}$ and $B\in\mathbb{R}^{4\times 2}$, which of the following is defined?

  1. $AB$ is $3\times 2$
  2. $BA$ is $2\times 3$
  3. $A+B$
  4. $AB^{\top}$ is $3\times 4$
Answer: A. Inner dimensions (4 and 4) match; the result inherits outer dimensions $3$ and $2$. $BA$ is not defined because $2\neq 3$. $A+B$ is not defined because shapes differ.

Let $A=\begin{pmatrix}1&2\\3&4\end{pmatrix}$ and $B=\begin{pmatrix}0&1\\1&0\end{pmatrix}$. Compute $AB$, $BA$, and verify that $(AB)^{\top}=B^{\top}A^{\top}$.

A dataset $X\in\mathbb{R}^{1000\times 50}$ is mean-centred and multiplied by a weight matrix $W\in\mathbb{R}^{50\times 10}$ to produce logits $Z=XW$. What is the shape of $Z$? How many scalar multiplications does computing $Z$ take?

Recap Card

  • Matrix $\times$ vector = linear combination of columns.
  • Inner dimensions must match for multiplication.
  • Not commutative, but associative and distributive.
  • Special shapes (diagonal, symmetric, orthogonal) appear everywhere in ML.

Chapter Three · Solving $A\mathbf{x}=\mathbf{b}$

Systems of Linear Equations

A linear system is a set of $m$ equations in $n$ unknowns. In matrix form it is $A\mathbf{x}=\mathbf{b}$. Three outcomes are possible: one unique solution, infinitely many, or none. Which case you are in depends entirely on how the columns of $A$ span the right-hand side $\mathbf{b}$.

3.1 Row-echelon and reduced row-echelon form

Definition · Row Echelon Form (REF)
A matrix is in row echelon form if: (i) all-zero rows are at the bottom, (ii) the leading (first non-zero) entry of each non-zero row — called a pivot — is to the right of the pivot in the row above. If, in addition, every pivot is $1$ and is the only non-zero entry in its column, the matrix is in reduced row echelon form (RREF).

3.2 Gaussian elimination

Three elementary row operations preserve the solution set: (1) swap two rows, (2) multiply a row by a non-zero scalar, (3) add a multiple of one row to another. Apply them to the augmented matrix $[A\,|\,\mathbf{b}]$ until REF/RREF is reached, then back-substitute.

3.3 Worked example

Solve $$\begin{aligned} x+2y+z&=4\\ 2x+5y+3z&=9\\ -x+y+2z&=5.\end{aligned}$$ After $R_2\!\leftarrow\!R_2-2R_1$ and $R_3\!\leftarrow\!R_3+R_1$: $$\begin{pmatrix}1&2&1 \\ 0&1&1 \\ 0&3&3\end{pmatrix}\mathbf{x}=\begin{pmatrix}4\\1\\9\end{pmatrix}.$$ Then $R_3\!\leftarrow\!R_3-3R_2$ gives a zero row with right-hand side $6$ — a contradiction, so the system is inconsistent. (If the RHS were $3$ instead we would have infinitely many solutions.)

3.4 Existence and uniqueness

SituationConditionPicture
Unique solutionrank$(A)=$ rank$(A|\mathbf{b})=n$Lines intersect at a single point
Infinitely manyrank$(A)=$ rank$(A|\mathbf{b})< n$Lines coincide / planes share a line
No solutionrank$(A)<$ rank$(A|\mathbf{b})$Parallel lines / skew planes
Data Scientist Lens
When a dataset has more features than samples ($n>m$), the system $X\boldsymbol{\beta}=\mathbf{y}$ is typically underdetermined — infinitely many $\boldsymbol{\beta}$ fit. Regularisation (Ridge, Lasso) picks one specific solution by adding a constraint. 
import numpy as np
A = np.array([[1, 2, 1], [2, 5, 3], [-1, 1, 2]], float)
b = np.array([4, 9, 5], float)
# least-squares always returns *something*; check residual:
x, residuals, rank, sv = np.linalg.lstsq(A, b, rcond=None)
print(rank)        # < 3 would indicate deficiency
print(np.linalg.matrix_rank(np.c_[A, b]))  # compare augmented rank

Q. A system $A\mathbf{x}=\mathbf{b}$ has rank$(A)=2$ and rank$([A|\mathbf{b}])=3$. Which is true?

  1. Unique solution
  2. Infinitely many solutions
  3. No solution
  4. Cannot determine without $A$
Answer: C. The augmented matrix has higher rank than $A$, so $\mathbf{b}$ is not in the column space of $A$ — the system is inconsistent.

Solve by Gaussian elimination: $x+y-z=2,\;\;2x-y+z=3,\;\;x-2y+2z=1$.

For which values of $k$ does the system $x+y=1,\;\;2x+ky=2$ have (i) a unique solution, (ii) infinitely many, (iii) none?

Recap Card

  • Elementary row operations never change the solution set.
  • Pivot positions tell you rank.
  • Rank of $A$ vs rank of $[A|\mathbf{b}]$ determines existence / uniqueness.
  • Underdetermined systems are the norm in high-dimensional ML.

Chapter Four · The number that reveals shape

Determinants & Invertibility

The determinant $\det(A)$ of a square matrix is a single number that encodes how $A$ scales (signed) volume. When $\det(A)=0$, the matrix squashes space onto a lower dimension; it cannot be inverted. When $|\det(A)|$ is small but non-zero, inversion is still possible but numerically dangerous.

4.1 $2\times 2$ and $3\times 3$ determinants

For $A=\begin{pmatrix}a&b\\c&d\end{pmatrix}$, $\det(A)=ad-bc$. For $3\times 3$, use the rule of Sarrus or cofactor expansion along any row/column:

$$\det(A)=\sum_{j=1}^{n}(-1)^{i+j}a_{ij}M_{ij},$$(4.1)

where $M_{ij}$ is the minor obtained by deleting row $i$ and column $j$.

4.2 Properties

  • $\det(I)=1$, $\det(A^{\top})=\det(A)$, $\det(AB)=\det(A)\det(B)$.
  • Swapping two rows changes the sign; scaling a row by $\alpha$ scales $\det$ by $\alpha$; adding a multiple of one row to another leaves $\det$ unchanged.
  • For triangular matrices, $\det$ is the product of diagonal entries.

4.3 Geometric meaning

4.4 The Invertibility Theorem

Theorem · Equivalent conditions for invertibility
For a square matrix $A\in\mathbb{R}^{n\times n}$, the following are all equivalent:
  1. $A$ is invertible (i.e., $A^{-1}$ exists).
  2. $\det(A)\neq 0$.
  3. rank$(A)=n$ (columns are linearly independent).
  4. $A\mathbf{x}=\mathbf{0}$ only when $\mathbf{x}=\mathbf{0}$.
  5. $A\mathbf{x}=\mathbf{b}$ has a unique solution for every $\mathbf{b}$.
  6. $0$ is not an eigenvalue of $A$.

If $A^{-1}$ exists, Cramer's formula gives $\mathbf{x}=A^{-1}\mathbf{b}$, but in practice you should almost never compute $A^{-1}$ directly — solve the linear system instead (Chapter 10).

Data Scientist Lens
A covariance matrix $\Sigma$ is singular when features are linearly dependent — a classic cause of the "singular matrix" error when fitting linear regression without regularisation. Ridge regression adds $\lambda I$ to guarantee invertibility: $(X^{\top}X+\lambda I)^{-1}$. 
A = np.array([[1, 2], [3, 4]], float)
print(np.linalg.det(A))        # -2.0
print(np.linalg.inv(A))        # [[-2,  1], [ 1.5, -0.5]]
# Prefer solve() over inv() for Ax=b:
b = np.array([5, 11])
print(np.linalg.solve(A, b))   # [1, 2]

Q. If $A\in\mathbb{R}^{n\times n}$ and $\det(A)=0$, which statement must be true?

  1. $A$ has at least one zero row.
  2. The columns of $A$ are linearly dependent.
  3. $A$ is the zero matrix.
  4. $A=A^{\top}$.
Answer: B. A zero determinant is equivalent to rank-deficiency, which is equivalent to the columns being linearly dependent. (A) and (C) are too strong; (D) is unrelated.

Compute $\det\begin{pmatrix}2&1&3\\0&-1&4\\1&2&0\end{pmatrix}$ by cofactor expansion along the first column.

Find all values of $\lambda$ for which the matrix $A-\lambda I$ is singular, where $A=\begin{pmatrix}3&1\\0&2\end{pmatrix}$. (Preview of eigenvalues!)

Recap Card

  • $\det(A)=0\Leftrightarrow A$ is not invertible $\Leftrightarrow$ columns dependent.
  • Determinants multiply: $\det(AB)=\det(A)\det(B)$.
  • Geometrically $|\det A|$ is a volume scale factor.
  • Never compute $A^{-1}$ in practice — solve the system instead.

Chapter Five · Structure of solutions

Basis, Dimension & Rank

Every matrix defines four natural subspaces. Understanding them explains why a regression model has a unique solution, when a least-squares fit exists, and where redundant features hide.

5.1 Linear independence and span

Definition · Linear independence
Vectors $\mathbf{v}_1,\dots,\mathbf{v}_k$ are linearly independent if $c_1\mathbf{v}_1+\dots+c_k\mathbf{v}_k=\mathbf{0}$ implies $c_1=\dots=c_k=0$. Their span is the set of all linear combinations — a subspace of $\mathbb{R}^{n}$.

5.2 Basis and dimension

A basis of a subspace is an independent spanning set. Every vector in the subspace has a unique expansion in that basis. The number of basis vectors is the dimension — an intrinsic property that does not depend on which basis you pick.

5.3 The four fundamental subspaces

SubspaceDefinitionLives inDimension
Column space $C(A)$span of columns$\mathbb{R}^{m}$$r$
Null space $N(A)$$\{\mathbf{x}:A\mathbf{x}=\mathbf{0}\}$$\mathbb{R}^{n}$$n-r$
Row space $C(A^{\top})$span of rows$\mathbb{R}^{n}$$r$
Left null space $N(A^{\top})$$\{\mathbf{y}:A^{\top}\mathbf{y}=\mathbf{0}\}$$\mathbb{R}^{m}$$m-r$
Theorem · Rank–Nullity
For $A\in\mathbb{R}^{m\times n}$, $\;\text{rank}(A)+\dim N(A)=n$.

5.4 Worked example

Let $A=\begin{pmatrix}1&2&3\\2&4&6\end{pmatrix}$. Row reducing gives a single pivot, so rank$=1$. The null space is $\{(x,y,z):x+2y+3z=0\}$, two-dimensional (a plane). Check: $1+2=3=n$.

Data Scientist Lens
In a design matrix $X$, perfectly collinear features (e.g., temperature in °C and °F) live in the row space and make rank$(X)< n$. Their coefficient vector lies in the null space — no unique least-squares solution. Dropping one of the collinear features is the cleanest fix. 
A = np.array([[1,2,3],[2,4,6]], float)
print(np.linalg.matrix_rank(A))      # 1
# Null space via SVD:
U, S, Vt = np.linalg.svd(A)
ns = Vt[A.shape[0]:].T                # columns span N(A)
print(ns.shape, "  A @ ns ≈", np.allclose(A @ ns, 0))

Q. $A\in\mathbb{R}^{5\times 7}$ has rank $4$. What is $\dim N(A)$?

  1. $1$
  2. $3$
  3. $4$
  4. $7$
Answer: B. By rank–nullity, $\dim N(A)=n-r=7-4=3$.

Find a basis for the column space and null space of $A=\begin{pmatrix}1&0&1\\0&1&1\\1&1&2\end{pmatrix}$.

Show that if $\mathbf{v}\in N(A)$ and $\mathbf{u}\in C(A^{\top})$, then $\mathbf{u}^{\top}\mathbf{v}=0$.

Recap Card

  • Rank = pivots = dimension of row / column space.
  • Rank–nullity: columns minus pivots $=$ nullity.
  • Null space $\perp$ row space; left null $\perp$ column space.
  • Linear dependence among features = non-trivial null space.

Chapter Six · Matrices as functions

Linear Transformations

A linear transformation $T:\mathbb{R}^{n}\to\mathbb{R}^{m}$ satisfies $T(\alpha\mathbf{u}+\beta\mathbf{v})=\alpha T(\mathbf{u})+\beta T(\mathbf{v})$. Once you fix bases, every linear transformation is captured by a unique matrix.

6.1 The matrix of a transformation

If $T(\mathbf{e}_j)=\mathbf{a}_j$ for the $j$-th standard basis vector, then $T(\mathbf{x})=A\mathbf{x}$ with $A=[\mathbf{a}_1\,|\,\mathbf{a}_2\,|\,\dots\,|\,\mathbf{a}_n]$. So: apply $T$ to each basis vector and stack the results as columns.

6.2 Gallery of 2-D linear maps

Common 2-D matrices:

$$R_{\theta}=\begin{pmatrix}\cos\theta&-\sin\theta\\ \sin\theta&\cos\theta\end{pmatrix},\quad S_{a,b}=\begin{pmatrix}a&0\\0&b\end{pmatrix},\quad H_{k}=\begin{pmatrix}1&k\\0&1\end{pmatrix}.$$

6.3 Composition and change of basis

If $T=T_1\circ T_2$ then the matrix of $T$ is the product of matrices in the same order: $A=A_1A_2$. Given a new basis $B$, the matrix of $T$ in that basis is $[T]_B=P^{-1}AP$ where $P$ is the change-of-basis matrix with $B$'s vectors as columns. We will see $P^{-1}AP$ again in Chapter 8 — it is the essence of diagonalisation.

6.4 Kernel and image

$\ker(T)=\{\mathbf{x}:T(\mathbf{x})=\mathbf{0}\}=N(A)$ and $\text{im}(T)=C(A)$. $T$ is injective iff $\ker(T)=\{\mathbf{0}\}$, surjective iff $\text{im}(T)=\mathbb{R}^{m}$, and invertible iff both, which forces $m=n$.

Data Scientist Lens
A dense layer $W$ is a linear transformation; a non-linear activation $\sigma$ is what lets a neural network escape pure linearity. Without $\sigma$, a 100-layer MLP would collapse into a single matrix $W=W_{100}\cdots W_1$ — no more expressive than a one-layer linear model. 
theta = np.deg2rad(30)
R = np.array([[np.cos(theta), -np.sin(theta)],
              [np.sin(theta),  np.cos(theta)]])
S = np.diag([2.0, 0.5])
T = R @ S                   # first scale, then rotate
print(T @ np.array([1, 0])) # image of e1

Q. Which of these functions is not a linear transformation?

  1. $T(\mathbf{x})=A\mathbf{x}$ with $A\in\mathbb{R}^{m\times n}$
  2. $T(\mathbf{x})=2\mathbf{x}$
  3. $T(\mathbf{x})=A\mathbf{x}+\mathbf{b}$ with fixed $\mathbf{b}\neq\mathbf{0}$
  4. $T(\mathbf{x})=0$
Answer: C. Affine maps fail the zero-preservation test: $T(\mathbf{0})=\mathbf{b}\neq\mathbf{0}$. (A), (B), (D) are all linear.

Write the $2\times 2$ matrix of the transformation that first rotates by $90^{\circ}$ counter-clockwise and then reflects across the $y$-axis. Apply it to $(1,0)^{\top}$.

Let $T:\mathbb{R}^{3}\to\mathbb{R}^{2}$ be $T(x,y,z)=(x+y,\,y-z)$. Write its matrix and describe $\ker(T)$ and $\text{im}(T)$.

Recap Card

  • Linear $\Leftrightarrow$ respects addition and scaling; fixes origin.
  • The matrix of $T$ has $T(\mathbf{e}_j)$ as its $j$-th column.
  • Composition $\leftrightarrow$ matrix product (order matters).
  • Kernel = null space, image = column space.

Chapter Seven · From geometry to regression

Orthogonality, Projections & Least Squares

Linear regression is the single most-used model in data science, and it is projection in disguise. When $A\boldsymbol{\beta}=\mathbf{y}$ has no exact solution, we instead find the $\boldsymbol{\beta}$ whose prediction $A\boldsymbol{\beta}$ is the closest point to $\mathbf{y}$ inside the column space of $A$ — the orthogonal projection.

7.1 Orthogonal sets & Gram–Schmidt

A set $\{\mathbf{q}_1,\dots,\mathbf{q}_k\}$ is orthonormal if $\langle\mathbf{q}_i,\mathbf{q}_j\rangle=\delta_{ij}$. Any independent set can be orthonormalised:

$$\mathbf{u}_k=\mathbf{v}_k-\sum_{j<k}\langle\mathbf{v}_k,\mathbf{q}_j\rangle\mathbf{q}_j,\qquad \mathbf{q}_k=\mathbf{u}_k/\|\mathbf{u}_k\|.$$

7.2 Orthogonal projection

If the columns of $A$ are independent, the projection of $\mathbf{y}$ onto $C(A)$ is

$$P=A(A^{\top}A)^{-1}A^{\top},\qquad \hat{\mathbf{y}}=P\mathbf{y}.$$(7.1)

$P$ satisfies $P=P^{\top}$ and $P^{2}=P$ — it is idempotent and symmetric.

7.3 The normal equations (linear regression)

Theorem · Least squares
The $\boldsymbol{\beta}$ that minimises $\|\mathbf{y}-A\boldsymbol{\beta}\|^{2}$ satisfies the normal equations $$A^{\top}A\,\boldsymbol{\beta}=A^{\top}\mathbf{y}.$$ If $A$ has full column rank, the unique solution is $\boldsymbol{\beta}=(A^{\top}A)^{-1}A^{\top}\mathbf{y}$.

7.4 Worked example

Fit $y=\beta_0+\beta_1 x$ to the points $(1,1),(2,2),(3,2)$.

$A=\begin{pmatrix}1&1\\1&2\\1&3\end{pmatrix},\;\mathbf{y}=\begin{pmatrix}1\\2\\2\end{pmatrix}.$ Then $A^{\top}A=\begin{pmatrix}3&6\\6&14\end{pmatrix}$, $A^{\top}\mathbf{y}=\begin{pmatrix}5\\11\end{pmatrix}.$ Solving gives $\hat{\boldsymbol{\beta}}=(2/3,\,1/2)^{\top}$, so $\hat y = 2/3 + x/2$.

Data Scientist Lens
Scikit-learn's LinearRegression().fit() solves exactly equation (7.1) under the hood (usually via QR or SVD, not by literally inverting $A^{\top}A$ — see Chapters 10–11). Ridge: $(A^{\top}A+\lambda I)\boldsymbol{\beta}=A^{\top}\mathbf{y}$. 
X = np.array([[1,1],[1,2],[1,3]], float)
y = np.array([1,2,2], float)
beta, *_ = np.linalg.lstsq(X, y, rcond=None)
print(beta)                # [0.666..., 0.5]
y_hat = X @ beta
residual = y - y_hat
print(X.T @ residual)      # ≈ 0 (orthogonality)

Chapter Eight · Hidden axes of a matrix

Eigenvalues & Eigenvectors

Some vectors are stretched but not rotated by a matrix; they are its eigenvectors, and the stretch factors are the eigenvalues. These hidden axes diagonalise the matrix and reveal its long-run behaviour.

Definition · Eigenvalue / Eigenvector
A non-zero $\mathbf{v}\in\mathbb{R}^{n}$ is an eigenvector of $A$ with eigenvalue $\lambda$ if $A\mathbf{v}=\lambda\mathbf{v}$. Equivalently, $(A-\lambda I)\mathbf{v}=\mathbf{0}$, so $\det(A-\lambda I)=0$.

8.1 The characteristic polynomial

$p_A(\lambda)=\det(A-\lambda I)$ is a polynomial of degree $n$. Its roots are the eigenvalues. Eigenvectors are found by solving $(A-\lambda I)\mathbf{v}=\mathbf{0}$.

8.2 Diagonalisation

If $A$ has $n$ independent eigenvectors $\mathbf{v}_1,\dots,\mathbf{v}_n$, stacking them as columns of $V$ gives

$$A=V\Lambda V^{-1},\quad \Lambda=\text{diag}(\lambda_1,\dots,\lambda_n).$$

Then $A^{k}=V\Lambda^{k}V^{-1}$, which is why eigendecomposition diagonalises powers of matrices (Markov chains, PageRank, recurrences).

8.3 Symmetric matrices — the best case

Spectral Theorem
Every real symmetric matrix $A=A^{\top}$ has real eigenvalues and an orthonormal basis of eigenvectors. It can be written $A=Q\Lambda Q^{\top}$ with $Q$ orthogonal.

This is why covariance matrices, kernel matrices, and Hessians — all symmetric — are so well-behaved.

8.4 Worked example

$A=\begin{pmatrix}4&1\\2&3\end{pmatrix}$. $\det(A-\lambda I)=(4-\lambda)(3-\lambda)-2=\lambda^{2}-7\lambda+10=(\lambda-5)(\lambda-2)$. Eigenvalues $5,2$. For $\lambda=5$: $(A-5I)\mathbf{v}=\mathbf{0}\Rightarrow \mathbf{v}=(1,1)^{\top}$. For $\lambda=2$: $\mathbf{v}=(1,-2)^{\top}$.

Data Scientist Lens
The power method — repeatedly applying $A$ to a random vector and normalising — converges to the top eigenvector. That is exactly how Google's original PageRank was computed on the web's hyperlink matrix. 
A = np.array([[4,1],[2,3]], float)
vals, vecs = np.linalg.eig(A)
print(vals)                 # [5. 2.]
# Verify A v = λ v:
for λ, v in zip(vals, vecs.T):
    print(np.allclose(A @ v, λ * v))

Chapter Nine · The Swiss-army knife

SVD & Principal Component Analysis

The singular value decomposition works for every matrix — rectangular, rank-deficient, noisy. It is the most important factorisation in applied mathematics and the mathematical backbone of PCA, latent-semantic analysis, collaborative filtering, and modern matrix completion.

Theorem · SVD
Every $A\in\mathbb{R}^{m\times n}$ can be written $$A=U\Sigma V^{\top},$$ with $U\in\mathbb{R}^{m\times m}$ orthogonal, $V\in\mathbb{R}^{n\times n}$ orthogonal, and $\Sigma\in\mathbb{R}^{m\times n}$ diagonal with non-negative singular values $\sigma_1\geq\sigma_2\geq\dots\geq 0$.

9.1 Intuition

9.2 Low-rank approximation (Eckart–Young)

Truncating the SVD to the top $k$ singular values, $$A_k=\sum_{i=1}^{k}\sigma_i\mathbf{u}_i\mathbf{v}_i^{\top},$$ gives the best rank-$k$ approximation of $A$ in the Frobenius norm. This is how every matrix-compression scheme (JPEG-style image compression, model distillation for dense layers, topic modelling) ultimately works.

9.3 Principal Component Analysis (PCA)

Given a centred data matrix $X\in\mathbb{R}^{m\times n}$:

  1. Compute SVD $X=U\Sigma V^{\top}$.
  2. The columns of $V$ are the principal directions (orthonormal features of maximum variance).
  3. The scores $XV=U\Sigma$ are the data rotated into the PC coordinate system.
  4. Variance explained by PC$_i$ is $\sigma_i^{2}/(m-1)$.

Equivalently, PCA diagonalises the covariance matrix $\Sigma_X=\frac{1}{m-1}X^{\top}X=V\Lambda V^{\top}$ with $\lambda_i=\sigma_i^{2}/(m-1)$.

Data Scientist Lens
Before training a model with thousands of correlated features, run PCA and keep enough components to explain ~95% variance. You usually get the same model accuracy with 10–20× fewer features and dramatically faster training. 
X = np.random.randn(200, 50)
X -= X.mean(0)
U, S, Vt = np.linalg.svd(X, full_matrices=False)
explained = (S**2) / (S**2).sum()
k = np.searchsorted(np.cumsum(explained), 0.95) + 1
scores = U[:, :k] * S[:k]                     # X @ V[:, :k]
reconstruction = scores @ Vt[:k]              # ≈ X for k large
print(k, "components retain 95% variance")

Recap Card

  • Every matrix: $A=U\Sigma V^{\top}$ — rotate, stretch, rotate.
  • Top-$k$ SVD = best rank-$k$ approximation (Eckart–Young).
  • PCA = SVD of centred data; variance = $\sigma_i^{2}/(m-1)$.
  • Compression, denoising, LSA, recsys — all leverage truncated SVD.

Chapter Ten · Practical solvers

Matrix Factorisations

In production you never solve $A\mathbf{x}=\mathbf{b}$ with pen-and-paper Gaussian elimination; you use a factorisation. Three dominate numerical practice: LU, QR, and Cholesky.

10.1 LU decomposition

Any (permuted) square $A$ admits $PA=LU$ with $L$ unit-lower-triangular and $U$ upper-triangular. Once factored, solve two triangular systems: $L\mathbf{y}=P\mathbf{b}$, then $U\mathbf{x}=\mathbf{y}$. Great when you need to solve many systems with the same $A$.

10.2 QR decomposition

Every $A\in\mathbb{R}^{m\times n}$ with $m\geq n$ admits $A=QR$ with $Q$ orthogonal columns and $R$ upper-triangular. For least-squares, $A\boldsymbol{\beta}=\mathbf{y}\Rightarrow R\boldsymbol{\beta}=Q^{\top}\mathbf{y}$. More numerically stable than normal equations.

10.3 Cholesky decomposition

If $A$ is symmetric positive-definite, $A=LL^{\top}$ with $L$ lower-triangular. Twice as fast as LU and the workhorse of Gaussian processes, Kalman filters, and Bayesian inference.

Chapter Eleven · When math meets floating point

Numerical Linear Algebra

On a computer, every number is rounded. Two mathematically identical algorithms can produce wildly different answers. This chapter explains the concepts every practitioner needs: conditioning, stability, and sparsity.

11.1 Condition number

Definition · Condition Number
$\kappa(A)=\sigma_{\max}(A)/\sigma_{\min}(A)$. If $\kappa$ is large, tiny perturbations of $\mathbf{b}$ can cause large perturbations of $\mathbf{x}=A^{-1}\mathbf{b}$.

Rule of thumb: if $\kappa(A)\approx 10^{k}$, expect to lose $\approx k$ digits of accuracy. Ridge regression and Tikhonov regularisation are methods of reducing the effective condition number.

11.2 Numerical stability: backward error

An algorithm is backward stable if it returns the exact solution to a slightly perturbed problem. Householder QR is backward stable; the textbook Gram–Schmidt is not — in floating point its "orthogonal" vectors drift. Use modified Gram–Schmidt, or (better) Householder reflections.

11.3 Sparse matrices

11.4 Iterative solvers

For huge sparse systems (millions of unknowns), direct factorisation is infeasible. Iterative methods — Conjugate Gradient (SPD), GMRES (general), Lanczos/Arnoldi (top-k eigenvalues) — converge in far fewer iterations than the matrix has rows, using only matrix-vector products.

Gotcha
Always check np.linalg.cond(A) before doing np.linalg.solve(A, b) on ill-conditioned problems. A condition number $>10^{12}$ in double precision essentially returns noise. 
from scipy.sparse import csr_matrix
from scipy.sparse.linalg import cg
A_sparse = csr_matrix(np.diag(np.arange(1, 1001, dtype=float)))
b = np.ones(1000)
x, info = cg(A_sparse, b, atol=1e-8)     # converges in O(√κ) steps
print(info)                               # 0 = converged

  • Iterative CG/GMRES scale where direct methods can't.

Chapter Twelve · Linear algebra in the wild

Applications in Data Science

This final chapter walks through four iconic data-science systems and pinpoints the linear algebra driving each.

12.1 Word embeddings & latent semantic analysis

Build the term–document matrix $X$, compute its truncated SVD $X_k=U_k\Sigma_k V_k^{\top}$. Rows of $U_k$ give a $k$-dimensional vector for every word; $\cos$-similarity in that space captures meaning. The same idea powers item2vec, node2vec, and classical collaborative filtering.

12.2 PageRank

Let $M$ be the column-stochastic adjacency matrix of the web. The PageRank vector $\mathbf{r}$ is the dominant eigenvector of the Google matrix $$G=\alpha M+(1-\alpha)\tfrac{1}{n}\mathbf{1}\mathbf{1}^{\top},\qquad \alpha\approx 0.85.$$ Power iteration converges in $\sim 50$ multiplications — cheap even for billions of pages because $M$ is extremely sparse.

12.3 Recommender systems (matrix factorisation)

Fit latent user and item matrices $U,V$ by minimising $\sum_{(i,j)\in \Omega}(R_{ij}-\mathbf{u}_i^{\top}\mathbf{v}_j)^{2}+\lambda(\|\mathbf{u}_i\|^{2}+\|\mathbf{v}_j\|^{2})$ over the observed entries $\Omega$ (alternating least squares, SGD, or implicit ALS).

12.4 Neural networks

Every forward pass of a feed-forward network is a sequence of matrix multiplies interleaved with non-linearities:

$$\mathbf{h}_{\ell}=\sigma(W_{\ell}\mathbf{h}_{\ell-1}+\mathbf{b}_{\ell}).$$

Back-propagation is repeated application of the chain rule — which in matrix form is transpose, multiply, and element-wise Hadamard product with $\sigma'$. Attention in Transformers computes $\text{softmax}(QK^{\top}/\sqrt{d})V$ — three matrix products. Training on a GPU is essentially a festival of dense matrix multiplications.

Data Scientist Lens · Tying it together
  • Regression → Chapter 7 (projections).
  • Logistic regression gradient → Chapter 1 (dot products) + Chapter 2.
  • PCA for viz → Chapter 9.
  • Recommender systems → Chapters 9 + 10 (low-rank, ALS).
  • Transformer attention → Chapters 2 + 8 (matrix products, eigenvalues for stability).
  • Gaussian processes → Chapter 10 (Cholesky).
import numpy as np

R = np.array([[5,3,0,1],
              [4,0,0,1],
              [1,1,0,5],
              [0,0,4,4]], float)
mask = (R > 0).astype(float)
k = 2; rng = np.random.default_rng(0)
U = rng.standard_normal((4, k)); V = rng.standard_normal((4, k))
